# NHL Faceoff ELO System

A research project to build an ELO rating system for NHL faceoffs.

## Overview

This project aims to analyze and rate NHL players' faceoff performance using an ELO-based rating system.

## Setup

```bash
# Create virtual environment
python -m venv venv

# Activate (Windows)
venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```
